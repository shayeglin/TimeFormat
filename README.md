# TimeFormat
This script allows you to format 12 hours and 24 hours back.

***

# 12-hour format
To convert the 12-hour format, the second parameter, '12' session.
Example :
`TimeFormat (15, '12') // 3`


***

# 24-hour format
To convert the 24-hour format, the second parameter, '24' session.
Example :
`TimeFormat (3, '24') // 15`
